برای اجرای سرور api می بایست مراحل زیر انجام شود :

  • پایتون بر روی سیستم عامل نصب باشد
  • وارد ترمینال یا cmd شده و فایل active مربوط به مجازی ساز فعال گردد
  • در همان cmd یا ترمینال مرحله قبل ، وارد پوشه برنامه حاوی manage.py شده و دستور زیر اجرا شود
  • python manage.py runserver
  • برای اجرای برنامه cmd یا ترمینال مرحله قبل را باز نگه دارید.
  • برای شروع برنامه بر روی فایل index.py واقع در پوشه اصلی برنامه کلیک کنید
  • نکته : آدرس api اجرایی http://127.0.0.1:8000/ می باشد.
